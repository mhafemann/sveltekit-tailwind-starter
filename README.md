# SvelteKit Starter

My personal SvelteKit Tailwindcss starter repo.

## Usage

Clone the repo:

```bash
npx degit mhafemann/sveltekit-tailwind-starter
```

Install dependecies:

```bash
npm install
```


## NPM Commands

Run in development mode:

```bash
npm run dev
```

Run development mode and serve on local network:

```bash
npm run dev:host
```
